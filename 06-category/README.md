# Övning 6: Kategorisering av böcker

## Instruktioner
- Låt `genre = "deckare"`.
- Om `genre` är "deckare": skriv "Spännande val!"
- Om "roman": skriv "En klassiker!"
- Om "facklitteratur": skriv "Lärorikt!"
- Annars: "Okänd kategori"

## Körning
`node index.js`

## Test
`npm install` && `npm test`
